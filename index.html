Robotic Handoff System

Programmed a mobile robot and robot arm to work together autonomously to simulate a radioactive materials handling process. The arm used inverse kinematics to pick blocks from precise cementation deposit locations and load them onto the mobile robot, which navigated a course and returned them to a safe handling cell. Built in ROS with the two systems communicating in real time. JEM 455 — Dr. Eli Buckner.

ROS C++ Inverse Kinematics Mobile Robotics Robot Arm

Quanser Aero Dynamic Systems Modeling

Derived and validated mathematical models for a dual propeller mechatronic system capable of controlled pitch and yaw motion. Applied KVL and Newton's rotational laws to build differential equations for the thruster, pitch, and yaw dynamics, then used system identification techniques to identify unknown parameters. All models were validated against real hardware data in MATLAB and Simulink. JEM 420 — Team Project.
